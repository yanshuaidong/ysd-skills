# ysd-news

获取财联社每日早间新闻精选，支持按日期范围查询，最多返回最近 10 天内的数据。

## 使用方法

运行 Python 脚本：

```bash
# 获取今天的早间新闻（默认）
python skills/ysd-news/fetch_news.py

# 获取指定日期范围的早间新闻
python skills/ysd-news/fetch_news.py --start_date 2026-03-20 --end_date 2026-03-24
```

脚本会输出 JSON 格式的新闻列表到 stdout，可直接读取解析。

## 参数说明

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| --start_date | 否 | 今天 | 起始日期，格式 YYYY-MM-DD |
| --end_date | 否 | 今天 | 结束日期，格式 YYYY-MM-DD |

## 日期限制

- 最早只能查询到 **当前日期往前 10 天** 的新闻
- 如果传入的日期超出范围，脚本会自动修正到有效范围内
- 结束日期不会超过今天

## 输出格式

JSON 数组，每条记录包含：

| 字段 | 说明 |
|------|------|
| id | 新闻 ID |
| title | 标题 |
| content | 新闻正文 |
| time | 发布时间 (YYYY-MM-DD HH:MM:SS) |
| created_at | 创建时间 |
| updated_at | 更新时间 |

## 示例输出

```json
[
  {
    "id": 62265,
    "title": "财联社3月24日早间新闻精选",
    "content": "【财联社3月24日早间新闻精选】\n1、习近平23日在河北雄安新区考察...\n2、美股周一开市前夕...",
    "time": "2026-03-24 08:22:23",
    "created_at": "2026-03-24 08:25:29",
    "updated_at": "2026-03-24 20:36:43"
  }
]
```

## 依赖

- Python 3（仅使用标准库，无需额外安装）

## 注意事项

- 本工具用于获取当天或最近几天的早间新闻，不适合历史数据回溯
- 如遇网络问题或 API 不可用，脚本会输出错误信息到 stderr 并以非零状态码退出
